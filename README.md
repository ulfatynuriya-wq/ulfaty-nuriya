# Islamic Application Documentation

## Features
- **Prayer Times**: Get accurate prayer times based on the user's location.
- **Quran Reading**: Access the entire Quran with translations and audio.
- **Islamic Calendar**: View and navigate through the Islamic calendar.
- **Dua & Azkar**: A collection of daily duas and azkar for remembrance.
- **Events**: Keep track of significant Islamic events and dates.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ulfaty-nuriya/your-repo-name.git
   cd your-repo-name
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Create a `.env` file on the root directory and add necessary configurations.

## Usage Guide
- Start the application:
   ```bash
   npm start
   ```
- Navigate through the app to explore features.

## Firebase Setup
1. Go to the [Firebase Console](https://console.firebase.google.com/).
2. Create a new project.
3. Add your app to the project and follow the instructions to configure it with Firebase.
4. Save the resulting configuration in your `.env` file.

## Deployment Steps
1. Build the application:
   ```bash
   npm run build
   ```
2. Deploy to your hosting service (e.g., Firebase Hosting, Vercel, etc.).
3. For Firebase Hosting, run:
   ```bash
   firebase deploy
   ```
